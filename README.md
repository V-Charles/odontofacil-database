# Banco de Dados — ERP Clínica Odontológica

Este repositório contém os **scripts SQL** do banco de dados utilizados pelo sistema **ERP para gestão de clínicas odontológicas**.  
O objetivo é centralizar toda a estrutura (DDL), dados iniciais (seeds) e eventuais migrações, facilitando **versionamento, manutenção e implantação** do banco de dados.

---

## Pré-requisitos

Para executar e testar este banco de dados, você precisará de:

- **MySQL Server** ≥ 8.0  
- **MySQL Workbench** (opcional, para visualização e execução gráfica)  
- **Acesso à linha de comando (`mysql`)** ou outro cliente compatível
